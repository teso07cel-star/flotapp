# 🛡️ Manual del Conductor Estratégico - FlotApp

Este protocolo detalla la operativa para conductores internos. El objetivo es la captura rápida de datos tácticos con mínima fricción.

---

## 🔑 1. Acceso y Registro

1. **Identificación**: Al abrir la app, seleccioná tu nombre de la lista de conductores autorizados.
2. **Inicio de Jornada**: Tocá el botón **"Iniciar Jornada"**. El sistema capturará automáticamente tu ubicación GPS de inicio.
3. **Hardware Corporativo**: Esta aplicación solo debe ejecutarse en dispositivos móviles autorizados. El intento de uso en hardware no laboral resultará en el bloqueo automático de la cuenta.

---

## 🚛 2. Reporte de Unidad (Primer Viaje)

La primera vez que utilices un vehículo en el día, el sistema requiere una validación completa:

1. **Patente**: Seleccioná o confirmá la patente de la unidad.
2. **Odómetro**: Ingresá el kilometraje actual que marca el tablero. Este paso es **obligatorio** para la sincronización inicial.
3. **Sucursales**: Seleccioná los nodos o sucursales de operativa que visitarás.
4. **Confirmación**: Tocá **"Confirmar Reporte"**.

---

## ⚡ 3. Modo Flash (Siguientes Reportes)

Para el resto de tus paradas o movimientos con la misma unidad, el flujo se acelera:

1. El sistema recordará tu patente y el odómetro validado.
2. **Ocultación de Campos**: El campo de kilometraje desaparecerá para evitar cargas redundantes.
3. **Acción**: Solo seleccioná las sucursales si han cambiado y tocá **"Confirmar Reporte"**.

---

## 🏁 4. Cierre de Turno

Al finalizar tu jornada laboral o al entregar la unidad:

1. Tocá el botón **"Cerrar Turno"**.
2. **Datos Finales**: Declará el **Kilometraje Final** y el **Nivel de Combustible**.
3. **Confirmación**: Tocá **"Confirmar Cierre"**. 

*Nota: A diferencia de otros perfiles, el Conductor Estratégico no necesita declarar el Lugar de Guarda.*

---

## ⚠️ Seguridad y Geolocalización

*   **Rastro Activo**: El rastro GPS permanece activo desde el Inicio de Jornada hasta el Cierre de Turno. 
*   **Bloqueo de Cuenta**: El sistema detecta intentos de falseo de ubicación o uso de emuladores. El bloqueo del dispositivo es irreversible y requiere intervención de sistemas.
